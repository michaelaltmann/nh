Distribution of Nursing Homes in Minnesota

This page is an experimental visualization of the availablility of nursing homes across Minnesota. {{> main}}

This application does not display correctly in Internet Explorer. The region around each nursing home is defined by the places for which that is the closest nursing home. The color indicates the average distance traveled by people in that catch basin to the nursing home, with darker red representing a larger value.
This application was developed using D3 and meteor with data from Google APIs. The code is available on GitHub
