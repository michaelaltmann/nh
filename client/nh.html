Emergency Department Quality for Minnesota Hospitals

This page is an experimental visualization of the data hospitals report to the federal goverment about the quality of care. The data are made available through Medicare. This visualization focuses on the measures related to the emergency department of MN hospitals. Select a quality measure to see a diagram of MN hospitals, colored by their emergency department quality. {{> main}}

This application does not display correctly in Internet Explorer. The region around each hospital is defined by the places for which that is the closest hospital. The color indicates the result of the measure, with darker red representing a larger value. Gray indicates that the hospital did not report a value for the selected measure.
This application was developed using D3 and meteor with data from Medicare and geocoding from Google APIs. The code is available on GitHub
